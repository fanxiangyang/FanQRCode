###  功能介绍

##### 功能介绍：封装系统自动二维码扫描与图片二维码解析 具体使用如下
* -(instancetype)initWithBlock:(ScanResultBlock)scanResultBlock;
* 注意：1.扫描是iOS7.0+，二维码图片解析iOS8.0+

###开发环境

* OS X 10.11beat
* Xcode Version 6.4
